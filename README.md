# Deutsch Lernen Docs - 德语学习文档中心

## 📚 项目概述

本仓库是 Discord 德语学习系统的完整文档中心，采用混合架构：
- **GitHub**：存放结构化、版本控制的完整文档
- **Discord**：保留导航、讨论和实时互动

## 🏗️ 文档结构

```
deutsch-lernen-docs/
├── docs/                    # 文档目录
│   ├── index.md            # 文档首页
│   ├── getting-started/    # 入门指南
│   ├── channels/           # 频道使用指南
│   ├── learning/           # 学习方法
│   ├── tools/              # 工具指南
│   └── resources/          # 学习资源
├── README.md              # 项目说明
└── mkdocs.yml            # 网站配置
```

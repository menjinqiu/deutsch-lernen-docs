# 德语知识点网络系统

## 🎯 系统概述

**德语知识点网络系统**是基于师父的 Notion 学习笔记构建的智能化知识管理系统。系统自动从师父的 Notion 笔记中提取知识点，构建结构化的知识网络，并提供智能化的学习支持。

### 📊 系统数据统计
- **数据来源**: Notion 德语学习笔记 (L1, L2, L3单元)
- **总计知识点**: 205 个
- **覆盖单元**: L1, L2, L3
- **知识分类**: 词汇、语法、发音、表达、练习
- **最后更新**: 2026年4月9日

### 🔗 知识网络维度
1. **单元维度** - 按学习进度组织 (L1 → L2 → L3)
2. **分类维度** - 按知识类型组织 (词汇、语法、发音等)
3. **关联维度** - 知识点之间的逻辑联系
4. **难度维度** - 基于CEFR等级的知识点分级

## 🔧 智能学习功能

### 1. 知识点检索系统
- **语义搜索**: 基于知识点的语义相似度搜索
- **分类过滤**: 按单元、分类、难度过滤知识点
- **关联推荐**: 推荐相关知识点和练习

### 2. 学习进度追踪
- **掌握程度评估**: 基于练习表现评估知识点掌握程度
- **遗忘曲线管理**: 基于艾宾浩斯遗忘曲线的复习安排
- **弱点分析**: 识别高频错误知识点

### 3. 个性化练习生成
- **针对性练习**: 基于弱点分析生成专项练习
- **情景模拟**: 基于知识点的情景对话练习
- **综合测试**: 单元综合测试和模拟考试

### 4. 知识网络可视化
- **知识图谱**: 可视化展示知识点之间的关联
- **学习路径**: 个性化学习路径推荐
- **进度热图**: 学习进度和掌握程度可视化

## 🚀 使用指南

### 快速开始
1. **查看单元知识**: 选择对应的单元页面查看详细知识点
2. **按分类学习**: 选择知识分类页面进行专项学习
3. **智能练习**: 使用智能练习功能巩固知识点

### 学习建议
1. **系统学习**: 按单元顺序系统学习知识点
2. **分类强化**: 针对薄弱分类进行专项强化
3. **定期复习**: 利用智能复习系统定期回顾
4. **实践应用**: 通过情景练习应用所学知识

### 数据同步
- **自动同步**: 系统定期从 Notion 同步最新笔记
- **手动更新**: 可在 Notion 中添加新知识点后手动触发同步
- **版本管理**: 所有知识点变更都有版本记录

## 📚 发音规则分布

### 单元 L1 发音规则 (10个)

- **元音有以下**: 德语元音发音体系概述

#### 单个辅音发音 (6个)
1. **e的发音规则**: 字母e在不同位置的发音
2. **s的发音规则**: s在不同位置的发音差异
3. **h的发音规则**: h的发音特点
4. **z的发音规则**: z的标准发音
5. **f的发音规则**: f的发音要领
6. **g的发音规则**: g在不同位置的发音变化

#### 字母组合发音 (3个)
1. **【ao】的发音**: ao组合的发音
2. **【ai】的发音**: ai组合的发音
3. **r的发音**: 德语r的特色发音

## 🧠 学习策略

### 元音学习策略
1. **对比学习**: 对比相似元音的差异
2. **长度练习**: 练习长短元音的区分
3. **口型模仿**: 模仿正确口型和舌位
4. **录音对比**: 录音后与标准发音对比

### 辅音学习策略
1. **部位感知**: 感知发音部位和方法
2. **清浊对比**: 对比清浊辅音的差异
3. **组合练习**: 练习辅音组合的发音
4. **词位练习**: 练习词首、词中、词尾发音

### 综合学习策略
1. **系统学习**: 按发音体系系统学习
2. **多感官学习**: 结合听、看、说、感学习
3. **渐进练习**: 从慢速到快速渐进练习
4. **情景应用**: 在真实交流中应用

## 🧪 发音练习系统

### 音素练习
1. **跟读练习**: 跟随音频模仿单个音素
2. **对比练习**: 对比相似音素的差异
3. **辨音练习**: 听音辨别不同音素
4. **发音练习**: 练习正确发音方法

### 音节练习
1. **音节拆分**: 练习单词的音节拆分
2. **音节重读**: 练习音节的重读规则

4. **节奏练习**: 练习德语的节奏模式

3. **单词应用**: 在句子中应用单词发音
4. **错误纠正**: 分析纠正单词发音错误

### 句子练习
1. **句子跟读**: 跟随音频模仿句子发音
2. **语调练习**: 练习句子的语调模式
3. **连读练习**: 练习句子中的连读技巧
4. **情景对话**: 情景对话中的发音应用

### 测试评估
1. **音素测试**: 单个音素发音测试
2. **单词测试**: 单词发音准确性测试
3. **句子测试**: 句子发音流畅性测试
4. **情景测试**: 情景交流中的发音测试

## 📈 掌握程度评估

### 评估标准
- **初级掌握**: 60%的音素发音基本正确
- **中级掌握**: 80%的单词发音准确自然
- **高级掌握**: 95%的发音接近母语水平

### 评估维度
1. **准确性**: 音素发音的准确程度
2. **清晰度**: 发音的清晰可懂程度
3. **流畅性**: 发音的自然流畅程度
4. **语调**: 语调和节奏的恰当程度

### 常见发音问题
1. **元音问题**: 长短不分、口型不对
2. **辅音问题**: 清浊混淆、部位错误
3. **组合问题**: 字母组合发音错误
4. **语调问题**: 语调模式不自然

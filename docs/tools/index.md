## 📚 词典和翻译工具

### DeepL 翻译
**特点**：
- 翻译质量高，语境理解好
- 支持文档翻译
- 有桌面和网页版

**使用技巧**：
1. **整句翻译**：输入完整句子而非单词
2. **对比学习**：中德对照学习表达差异
3. **文档学习**：翻译德语文章学习表达

**最佳实践**：
- 用于写作辅助和表达学习
- 避免过度依赖，注重理解
- 学习翻译中的地道表达

### Leo Dict 词典
**特点**：
- 专业德语词典
- 丰富例句和用法
- 发音和变位信息

**使用技巧**：
1. **单词查询**：查询详细释义和用法
2. **例句学习**：通过例句理解单词用法
3. **发音学习**：跟读标准发音

### Dict.cc 词典
**特点**：
- 用户贡献内容
- 实际使用例句
- 短语和习惯用法

## 🔧 语法检查工具

### LanguageTool
**功能**：
- 德语语法检查
- 拼写纠正
- 风格建议

**集成方式**：
- 浏览器扩展
- 桌面应用
- 在线编辑器

**使用建议**：
- 写作时实时检查
- 学习错误类型

### Duden Mentor
**特点**：
- 基于 Duden 权威词典
- 详细的错误解释
- 学习建议

## 🎤 发音练习工具

### Forvo
**功能**：
- 母语者发音库
- 单词和短语发音
- 不同地区口音

**使用技巧**：
1. **跟读练习**：模仿母语者发音
2. **对比学习**：比较不同发音
3. **录音对比**：录制自己的发音对比

### Google 翻译发音
**特点**：
- 免费易用
- 文本转语音
- 句子级发音

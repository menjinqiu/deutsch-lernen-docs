# 主频道使用指南

📖-德语-主频道是 Discord 德语学习系统的核心交流平台。

## 🎯 频道定位

### 核心功能
- **学习指挥中心**：每日学习计划发布和讨论
- **进度汇报平台**：学习进度汇报与反馈
- **综合问题讨论区**：综合问题深度讨论
- **学习心得分享区**：学习心得与经验分享

### 使用频率
- **每日多次**：主要学习和交流频道
- **早上查看**：今日学习目标
- **学习中**：随时汇报进展
- **晚上**：学习总结

### 表情反应系统
使用标准化表情反应快速反馈：

| 表情 | 含义 | 使用场景 |
|------|------|---------|
| ✅ | 完全理解掌握 | 确认理解，任务完成 |
| 🤔 | 部分理解需复习 | 有些困惑，需要更多解释 |

| 💡 | 有新见解/发现 | 有新的理解或发现 |

| 📝 | 需要做笔记 | 重要内容需要记录 |
| ❓ | 有问题要问 | 标记需要解答的问题 |

## 🧵 线程使用指南

### 何时创建线程
**创建线程的情况**：
1. **深度讨论**：需要多轮讨论的具体问题
2. **专题学习**：某个语法点的系统学习
3. **资源整理**：相关资源的收集和讨论
4. **项目协作**：需要多人协作的学习任务

**不创建线程的情况**：
1. **简单问题**：可以快速回答的问题
2. **简短交流**：简单的确认或反馈
3. **日常汇报**：常规的学习进度汇报

### 线程管理
**线程命名规范**：
```
【主题】简要描述 - 日期
示例：【语法】动词变位规则讨论 - 2026-04-07
示例：【资源】德语学习网站推荐 - 2026-04-07
```

**线程归档**：
- 已解决的问题线程 → 每周日归档
- 长期讨论线程 → 每月整理
- 过时内容线程 → 标记后删除

## 🔔 通知设置

- @齐天大圣德语教练 的直接@提及

**可选接收的通知**：
- 常规的学习讨论

### 通知优化设置
**Discord 设置**：
1. 频道通知：所有消息
2. @提及通知：所有人
3. 关键词通知：添加学习相关关键词

**移动端设置**：
1. 重要消息推送：开启
2. 非工作时间免打扰：设置
3. 快速回复功能：启用

## 🆘 问题解决流程

### 学习问题
**解决步骤**：
1. **自助解决**：查看文档和过往讨论
2. **社区提问**：在主频道描述问题
3. **专业解答**：@齐天大圣德语教练 获取帮助
4. **记录解决**：将解决方案记录到笔记频道

**问题分类**：
- **技术问题**：工具使用、系统设置
- **学习问题**：语法理解、发音困难
- **方法问题**：学习策略、时间管理
- **资源问题**：材料选择、工具推荐

### 系统问题
**反馈渠道**：
1. **直接反馈**：在主频道描述问题
2. **功能请求**：使用特定格式提出
3. **错误报告**：详细描述错误情况

**问题优先级**：
- **紧急**：影响学习的关键问题
- **重要**：需要改进的重要功能
- **一般**：优化建议和小问题
- **未来**：长期改进建议

# 笔记共享频道使用指南

📝-德语-笔记共享是德语学习系统的知识管理系统。

## 🎯 频道定位

### 核心功能
- **知识整理中心**：个人学习笔记整理与共享
- **语法总结库**：语法规则系统化总结
- **单词记忆库**：单词记忆卡片制作和分享
- **错题记录本**：错题本与难点记录

### 使用频率
- **每日1-2次**：学习后整理笔记
- **按需使用**：查找和复习知识点
- **定期整理**：每周整理归档

## 📁 文件管理

### 文件命名规范
```
格式：日期_主题_级别_类型
示例：20240407_日常对话_A1_笔记.md
示例：20240407_动词变位_A2_语法.md
示例：20240407_学习总结_周报.md
```

### 文件类型
1. **Markdown 文件**：`.md` 格式，推荐使用
2. **图片文件**：`.png`, `.jpg` 格式，图表和截图
3. **PDF 文件**：`.pdf` 格式，扫描文档
4. **音频文件**：`.mp3` 格式，发音录音

## 🔄 笔记工作流程

### 日常笔记流程
**学习时**：
1. 实时记录重点内容
2. 标记不理解的部分
3. 收集相关例句

**学习后**：
1. 整理和归类笔记
2. 补充详细解释
3. 添加记忆技巧
4. 分享到笔记频道

**复习时**：
1. 定期回顾笔记
2. 更新掌握程度
3. 补充新的理解

### 协作笔记流程
**分享笔记**：
1. 使用标准格式
2. 添加清晰标签
3. 说明适用级别

**讨论笔记**：
1. 在笔记下回复讨论
2. 提出改进建议
3. 分享个人经验

**整合笔记**：
1. 合并相似主题笔记
2. 创建专题总结
3. 建立知识体系

## 🏷️ 标签系统

### 内容标签
**难度标签**：
- `#A1` - 初级入门
- `#A2` - 基础水平
- `#B1` - 中级水平
- `#B2` - 中高级水平
- `#C1` - 高级水平
- `#C2` - 精通水平

**类型标签**：
- `#词汇` - 单词和短语
- `#语法` - 语法规则
- `#听力` - 听力相关
- `#口语` - 口语表达
- `#阅读` - 阅读技巧
- `#写作` - 写作指导

**主题标签**：
- `#日常生活` - 日常对话
- `#工作学习` - 工作学习场景
- `#旅行交通` - 旅行交通相关
- `#文化社会` - 文化社会话题
- `#科技经济` - 科技经济内容

### 状态标签
**掌握程度**：
- `#已掌握` - 完全理解
- `#需复习` - 需要定期复习
- `#有困难` - 理解有困难
- `#待学习` - 计划学习内容

**笔记状态**：
- `#草稿` - 初步记录
- `#整理中` - 正在整理
- `#已完成` - 整理完成
- `#已归档` - 已归档整理

### 查找方法
**Discord 搜索**：
- 按标签搜索：`#词汇 #A1`

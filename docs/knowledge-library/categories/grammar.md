### 核心语法点

#### 1. 动词现在时变位
```
ich   -e      (我)
du    -st     (你)
er/sie/es -t (他/她/它)
wir   -en     (我们)
ihr   -t      (你们)
sie/Sie -en   (他们/您)
```

#### 2. 名词的性数格
- **性**: 阳性(der)、阴性(die)、中性(das)
- **数**: 单数、复数
- **格**: 第一格(主格)、第二格(属格)、第三格(与格)、第四格(宾格)

#### 3. 冠词使用规则
- **定冠词**: der, die, das (特定)
- **不定冠词**: ein, eine (不特定)
- **零冠词**: 某些情况不用冠词

#### 4. 句子结构
- **陈述句**: 主语 + 动词 + 其他
- **疑问句**: 动词 + 主语 + 其他
- **否定句**: 使用nicht或kein否定

## 🔤 **L1 单元基础词汇 (从你的笔记提取)**

### 1. 连接词
- **aber** - 但是
- **und** - 并且
- **oder** - 或者

### 2. 代词
- **Sie** - 您 (正式)
- **ich** - 我
- **du** - 你 (非正式)
- **Ihnen** - 您 (与格)
- **dir** - 你 (与格)

### 3. 动词
- **hören** - 听
- **sprechen** - 说
- **lesen** - 读
- **ergänzen** - 填写
- **an/kreuzen** - (填表，答题)打勾，勾选
- **weiß, weißt** - 知道
- **heißen** - 叫做
- **kommen** - 来
- **geht** - 过，进行

### 4. 问候语
- **Hallo!** - 你好！(搭讪，随意打招呼)
- **Guten Morgen!** - 早上好 (5-10a.m.)
- **Morgen!** - 早上好 (简写)
- **Guten Tag!** - 你好！(白天用)
- **Guten Abend** - 晚上好 (6-10p.m.)
- **Gute Nacht!** - 晚安

### 5. 告别语
- **Auf Wiedersehen!** - 期待再见 (正式)
- **Tschüss!** - Bye！(非正式)
- **Ciao** - 再见 (意大利语)
- **Bis morgen** - 明天见！
- **Bis gleich** - 一会见！
- **Bis bald** - 期待不久之后见

### 6. 其他实用词汇
- **bitte** - 请
- **was** - 什么
- **wo** - 哪里
- **wie** - 怎么样
- **danke** - 谢谢
- **gut** - 好
- **auch** - 也
- **sehr** - 非常
- **nicht** - 不
- **ja** - 是
- **nein** - 不是
- **aus** - 从...
- **woher** - 从哪？

### 7. 时间相关
- **Tag** - 白天，一天
- **Morgen** - 早晨，清晨
- **Abend** - 傍晚
- **Nacht** - 夜晚

### 8. 状态描述
- **richtig** - 正确
- **falsch** - 错误
- **schlecht** - 不好
- **Solala** - 一般般
- **angenehm** - 愉快的，惬意的

## 🏢 **L2 单元机场场景词汇 (从你的笔记提取)**

### 1. 机场基础词汇
- **der Flughafen** = 飞机场 (Flug + Hafen)
- **Frankfurt** = 法兰克福
- **Main** = 美茵河
- **der Zoll** = 海关
- **am Zoll** = 在海关

### 2. 职业相关词汇
- **der Beamter** = 官员，公务员 (男)
- **die Beamtin** = 官员，公务员 (女)
- **der Zollbeamter** = 海关官员 (男)
- **die Zollbeamtin** = 海关官员 (女)

*德语职业词汇规则：男性为阳性，女性为阴性，以 -in 结尾表示女性*

### 3. 旅行用品词汇
- **der Koffer** = 箱子
- **der Pass** = 护照

### 4. 动作和描述词汇
- **hören** = 听
- **bitte** = 请
- **und** = 和
- **beantworten** = 回答
- **der Text** = 文章 (第一格)
- **den Text** = 文章 (第四格)

### 5. 介词和冠词组合
- **im** = in dem (介词+冠词) 在什么的里面

### 6. 扩展机场词汇
- **die Ankunft** = 到达
- **die Abflug** = 起飞
- **der Flug** = 飞行，航班
- **die Flugnummer** = 航班号
- **das Gate** = 登机口
- **die Sicherheitskontrolle** = 安全检查
- **das Gepäck** = 行李
- **die Gepäckaufgabe** = 行李托运
- **die Bordkarte** = 登机牌
- **der Check-in** = 值机
- **die Wartezone** = 等候区
- **die Toilette** = 厕所
- **der Duty-free-Shop** = 免税店
- **die Information** = 问讯处
- **der Ausgang** = 出口
- **der Eingang** = 入口

## 🎯 **词汇学习重点**

### 1. 基础词汇重点
#### 必须掌握的词汇
- 问候语：Hallo, Guten Tag, Auf Wiedersehen
- 人称代词：ich, du, Sie
- 基本动词：hören, sprechen, lesen
- 连接词：aber, und, oder

#### 日常交流词汇
- 时间词汇：Tag, Morgen, Abend, Nacht
- 状态词汇：gut, schlecht, richtig, falsch
- 礼貌词汇：bitte, danke, sehr angenehm

### 2. 场景词汇重点
#### 机场场景
- 核心词汇：Flughafen, Zoll, Pass, Koffer
- 职业词汇：Beamter, Beamtin
- 动作词汇：einchecken, beantworten

#### 学校场景
- 学校类型：Schule, Universität, Gymnasium
- 人员词汇：Schüler, Student, Lehrer, Professor
- 学科词汇：Deutsch, Mathematik, Physik, Chemie

### 3. 词汇记忆技巧
#### 分类记忆法
- 按场景分类：机场、学校、日常
- 按词性分类：名词、动词、形容词
- 按主题分类：问候、告别、学习、旅行

#### 联想记忆法
- 词根联想：Flug + Hafen = Flughafen
- 场景联想：在机场会用到哪些词汇
- 对比联想：男性/女性职业词汇对比

#### 应用记忆法
- 造句应用：用新词汇造句
- 对话应用：在对话中使用新词汇
- 情景应用：在特定情景中应用词汇

## 📋 **词汇练习建议**

### 1. 基础练习
#### 词汇翻译
- 德语→中文：Hallo, Guten Tag, Auf Wiedersehen
- 中文→德语：你好，早上好，再见

#### 词汇分类
- 将词汇按词性分类：名词、动词、形容词
- 将词汇按场景分类：机场、学校、日常

### 2. 中级练习
#### 词汇填空
- Ich ___ Deutsch. (lernen)
- Das ist ___ Buch. (ein)
- Wo ist ___ Flughafen? (der)

#### 词汇造句
- 用指定词汇造句
- 在句子中使用新词汇
- 编写包含多个新词汇的对话

### 3. 高级练习
#### 词汇扩展
- 学习词汇的同义词
- 学习词汇的反义词
- 学习词汇的派生词

#### 词汇应用
- 编写场景对话
- 描述图片或场景
- 进行角色扮演练习

## 🏢 **机场场景词汇 (从你的笔记提取)**

### 1. 机场基础词汇
- **der Flughafen** = 飞机场 (Flug + Hafen = 飞行的港口)
- **Frankfurt** = 法兰克福
- **Main** = 美茵河
- **der Zoll** = 海关
- **am Zoll** = 在海关

### 2. 职业相关词汇
- **der Beamter** = 官员，公务员 (男)
- **die Beamtin** = 官员，公务员 (女)
- **der Zollbeamter** = 海关官员 (男) (Zoll + Beamter)
- **die Zollbeamtin** = 海关官员 (女) (Zoll + Beamtin)

*德语职业词汇规则：男性为阳性，女性为阴性，以 -in 结尾表示女性*

### 3. 旅行用品词汇
- **der Koffer** = 箱子
- **der Pass** = 护照

### 4. 动作和描述词汇
- **hören** = 听
- **bitte** = 请
- **und** = 和
- **beantworten** = 回答
- **der Text** = 文章 (第一格)
- **den Text** = 文章 (第四格)

### 5. 介词和冠词组合
- **im** = in dem (介词+冠词) 在什么的里面

### 6. 更多场景词汇
- **die Ankunft** = 到达
- **die Abflug** = 起飞
- **der Flug** = 飞行，航班
- **die Flugnummer** = 航班号
- **das Gate** = 登机口
- **die Sicherheitskontrolle** = 安全检查
- **das Gepäck** = 行李
- **die Gepäckaufgabe** = 行李托运
- **die Bordkarte** = 登机牌
- **der Check-in** = 值机
- **die Wartezone** = 等候区
- **die Toilette** = 厕所
- **der Duty-free-Shop** = 免税店
- **die Information** = 问讯处
- **der Ausgang** = 出口
- **der Eingang** = 入口

## 🔤 **德语复合词构成 (从你的笔记学习)**

### 1. 复合词构成规则
#### 基本模式：名词 + 名词 = 新名词
- **Flug** (飞行) + **Hafen** (港口) = **Flughafen** (飞机场)
- **Zoll** (海关) + **Beamter** (官员) = **Zollbeamter** (海关官员)

#### 性别规则
- 复合词的性别由最后一个名词决定
- Flughafen (der) ← Hafen (der)
- Zollbeamter (der) ← Beamter (der)

### 2. 常见复合词类型
#### 地点+功能
- Flughafen (飞机场)
- Bahnhof (火车站)
- Krankenhaus (医院)

#### 职业+地点
- Zollbeamter (海关官员)
- Polizist (警察)
- Lehrer (老师)

#### 物品+用途
- Koffer (箱子)
- Pass (护照)
- Ticket (票)

### 3. 复合词记忆技巧
1. **分解记忆**：将复合词分解为组成部分
2. **联想记忆**：通过组成部分联想含义
3. **分类记忆**：按类型分类记忆
4. **应用记忆**：在场景中应用记忆

## 💬 **机场场景对话 (基于你的笔记扩展)**

### 1. 值机对话
```
旅客: Guten Tag! Ich möchte einchecken.
      (您好！我想办理值机。)
工作人员: Guten Tag! Ihren Pass, bitte.
         (您好！请出示您的护照。)
旅客: Hier ist mein Pass.
      (这是我的护照。)
工作人员: Danke. Und Ihr Gepäck?
         (谢谢。您的行李呢？)
旅客: Ich habe nur Handgepäck.
      (我只有手提行李。)
```

### 2. 海关检查对话
```
海关官员: Guten Tag! Ihren Pass, bitte.
         (您好！请出示您的护照。)
旅客: Hier bitte.
      (给您。)
海关官员: Was haben Sie im Koffer?
         (您的箱子里有什么？)
旅客: Nur Kleidung und Geschenke.
      (只有衣服和礼物。)
海关官员: Alles klar. Gute Reise!
         (明白了。旅途愉快！)
```

### 3. 问路对话
```
旅客: Entschuldigung, wo ist das Gate 15?
      (不好意思，15号登机口在哪？)
工作人员: Gehen Sie geradeaus, dann rechts.
         (直走，然后右转。)
旅客: Wie weit ist es?
      (有多远？)
工作人员: Etwa 5 Minuten zu Fuß.
         (大约步行5分钟。)
旅客: Vielen Dank!
      (非常感谢！)
```

### 4. 航班信息询问
```
旅客: Entschuldigung, wann fliegt der Flug nach Berlin?
      (不好意思，飞往柏林的航班什么时候起飞？)
工作人员: Der Flug LH123 fliegt um 14:30.
         (LH123航班14:30起飞。)
旅客: Von welchem Gate?
      (从哪个登机口？)
工作人员: Vom Gate 22.
         (从22号登机口。)
```

## 📝 **语法重点 (从你的笔记提取)**

### 1. 介词+冠词组合
#### im = in dem
- 表示"在...里面"
- 示例：im Flughafen (在机场里)，im Koffer (在箱子里)

#### am = an dem  
- 表示"在...旁边/在"
- 示例：am Zoll (在海关)，am Fenster (在窗边)

#### zum = zu dem
- 表示"去..."
- 示例：zum Flughafen (去机场)，zum Gate (去登机口)

### 2. 名词的格变化
#### 第一格 (Nominativ)
- 主语：der Text (文章)
- 示例：Der Text ist interessant. (这篇文章很有趣。)

#### 第四格 (Akkusativ)
- 直接宾语：den Text
- 示例：Ich lese den Text. (我读这篇文章。)

### 3. 职业名词的性别
#### 男性职业 (阳性)
- der Beamter (官员)
- der Lehrer (老师)
- der Arzt (医生)

#### 女性职业 (阴性)
- die Beamtin (女官员)
- die Lehrerin (女老师)
- die Ärztin (女医生)

*规则：女性职业名词通常在男性形式后加 -in*
